# Changelog

## 0.0.2

- Added a side preview command for active LaTeX files.
- Added KaTeX rendering for inline math, display math, and common display math environments.
- Added readable previews for headings, paragraphs, figures, tables, and algorithm blocks.
- Added image placeholders with declared size information.
- Added double-click navigation from preview content back to the source file.
- Added sample-based formula rendering validation.
